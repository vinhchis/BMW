HTML5

HTML5 is a markup language used for structuring and preseting content on the World Wide Web.

HTML5 was the fifth and last major version of HTML that is a World Wide Web Consortium(WEC) recommentdation. The current specification is known as the HTML Living Standard and is maintained by a consortium of the major browser vendors (Apple, Google, Mozilla, and Mocrosoft), the Web Hypertext Application Technology Working Group (WHATWG).
